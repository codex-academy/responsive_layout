Responsive webpages

Responsive web pages respond to the available screen size and move the web page elements around to ensure a good user experience. By combining Media Queries & Layout techniques you can create a responsive web page, that display elements below each other if the page is narrow and next to each other if the page is wider.
  • Media queries help to control which CSS is active based on the width of the screen.
  • Using layout techniques such as Floats & Flexbox elements can be moved around on the web page.
